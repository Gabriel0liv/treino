# Exame Treino 1 — Sistema de Biblioteca Universitária

Este pacote é uma base de treino, não uma solução completa.
Os ficheiros têm `TODO` nos pontos onde deves implementar durante o treino.

Estrutura:

- `exercicio1-spring-soap/` — SOAP com Spring-WS, porta 8002
- `exercicio2-rest-jaxrs/` — REST com JAX-RS/Jersey, porta 8001
- `exercicio3-jaxws-mediador/` — SOAP com JAX-WS mediador, porta 8000
- `exercicio4-xslt/` — XML + XSLT para transformar empréstimos em HTML
- `SITES_AUTORIZADOS.md` — palavras-chave para pesquisa nos sites autorizados

## Como abrir no VS Code

1. Extrai este ZIP.
2. Abre o VS Code.
3. Vai a `File > Open Folder...`.
4. Abre a pasta do exercício que vais resolver, por exemplo `exercicio1-spring-soap`.
5. Abre o terminal integrado: `Terminal > New Terminal`.
6. Usa os comandos Maven indicados no README de cada exercício.

## Ordem recomendada

1. Resolver primeiro o XSD da questão 1.
2. Depois configurar o WSDL e o endpoint.
3. Depois passar ao REST da questão 2.
4. Depois fazer o mediador da questão 3.
5. Por fim, resolver o XSLT.
