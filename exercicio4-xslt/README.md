# Exercício 4 — XSLT

Ficheiros:

- `exercicio4.xml`: XML de exemplo.
- `loans.xsl`: folha XSLT a completar.

## Como testar no browser

Abre `exercicio4.xml` no browser. Ele tem a linha:

```xml
<?xml-stylesheet type="text/xsl" href="loans.xsl"?>
```

## O que completar

Em `loans.xsl`, completar o `xsl:for-each` e os `xsl:value-of` para apresentar todos os empréstimos numa tabela.
